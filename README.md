# parrotias-livestreaming-client


## Running Locally

**1. Clone Repo**

```bash
git clone https://github.com/Steelzen/parrotias-livestreaming-client.git
```

**2. Install Dependencies**

```bash
npm install
```

**3. Run App**

```bash
npm start
```
